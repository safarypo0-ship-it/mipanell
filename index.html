۱. ریپوی گیتهاب رو بسازید و فایل‌ها رو آپلود کنید

۲. به railway.app برید → New Project → Deploy from GitHub repo

۳. ریپوی خودتون رو انتخاب کنید — Railway خودکار Dockerfile رو شناسایی می‌کنه

۴. از Settings → Networking → Generate Domain دامنه عمومی بگیرید

۵. دامنه رو در بخش تنظیمات سرور وارد کنید
